## aptget

>우분투에서 직접 설치는 서버에서 다운-컴파일로 어렵고 복잡한일.. / 편리하게 다운-설치 위해서 웹스토어 역할을 한다.

```
sudo apt-get install apache2 //설치

sudo apt-get update; //업데이트
```
