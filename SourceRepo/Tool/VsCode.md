###### 8626_101

vs code 설정변경
-

Markdown 프리뷰를 검은색 바탕과 흰색글씨로 커스텀 변경하려고 함.

File -> Preferences -> Settings

우측 창에 3가지가 있다.

USER SETTINGS WORKSPACE SETTINGS , FOLDERSETTING

나는 폴더 세팅으로 지정했고,

좌측창에 변경하려는 종류를 선택한후

바로수정안된다(Cannot edit in read-only editor) 
-> 설정값 좌측에 연필 아이콘이 있느걸 누러서 우측창에서 수정가능.
