### Http errorcode

| 200 | OK, 에러 없이 전송 성공                                                    |
|-----|----------------------------------------------------------------------------|
| 404 | Not Found,요청한 문서를 찾을 수 없음                                       |
| 500 | Internal Server Error, 서버 내부 오류 . 웹 서버 요청사항을 수행할 수 없음. |
